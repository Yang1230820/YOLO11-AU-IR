# YOLO11-AU-IR: A Lightweight Segmentation Framework for Multi-Class UAV Detection in Infrared Imagery  
# YOLO11-AU-IR：基于红外图像的多类别无人机检测轻量级分割框架

---

## Introduction / 简介

**English:**  
YOLO11-AU-IR is a lightweight segmentation framework specifically designed for infrared UAV detection. It addresses challenges such as small target sizes, low thermal contrast, and limited computational resources. The framework introduces key innovations including the Efficient Adaptive Downsampling (EADown) module, the HeteroScale Attention Network (HSAN), and the Adaptive Threshold Focal Loss (ATFL).

**中文：**  
YOLO11-AU-IR 是一款专门针对红外无人机检测设计的轻量级分割框架，旨在解决小目标尺寸、低热对比度以及计算资源受限等问题。该框架包含三个关键创新模块：高效适应下采样模块（EADown）、异构尺度注意力网络（HSAN）和自适应阈值焦点损失（ATFL）。
